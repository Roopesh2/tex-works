\usubsection{Aim}
\begin{enumerate}
	\item To design a 5V Zener series regulator for a input voltage of +8 to +20V and maximum load current of 50mA
	\item Plot and its load regulation and line regulation and find percentage load regulation and line regulation.
\end{enumerate}

\usubsection{Design}
$I_L(max) = 50mA, \ I_L(min) = 10mA, \ I_Z(\max) = 20mA, \ I_Z(min) = 10mA, \ V_Z = 5.1V$ \ $V_{in}(min) = 8V, \ V_{in}(max) = 20V$

$V_s(max) = V_{in}(max) - V_Z = 20V - 5.1V = 14.9V$

\vspace*{.5cm}
At minimum load current:

$R_s{max} = \dfrac{V_s(max)}{I_L(min) + I_Z{max}} = \dfrac{14.9V}{10mA + 20mA} \approx 500 \Omega $

\vspace*{.5cm}
At maximum load current:

$R_s{min} = \dfrac{V_s(max)}{I_L(max) + I_Z{min}} = \dfrac{14.9V}{50mA + 10mA} \approx 250 \Omega $

\vspace*{.5cm}
$\therefore R_s = \sqrt{R_s{max}\times R_s{min}} = \sqrt{500\times 250} = 353 \Omega$
